## Baihu CMS
Baihu is a CMS/Portal extension for phpBB.
It is still in an eary development stage, so don't use on production environment!

## Installation
Download/clone and move the repository into the folder phpbb_root/ext/baihu/baihu

### Enable / Disable
Go to admin panel / tab customise / Manage extensions -> enable / disable GZO (CMS)
