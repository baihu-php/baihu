## Baihu CMS
Baihu CMS/Portal is an extension for phpBB. It is currently in early development stage. Only use for testing!

## Installation
Download/clone and move the repository into the folder phpbb_root/ext/baihu/baihu

### Enable / Disable
Go to admin panel / tab customise / Manage extensions -> enable / disable Baihu CMS
